- Different between authentication and authorization
- Vulnerabilities mechanisms in 2 ways:
+ Weak mechanism, fail to adequately protec against brute-force attacks
+ Logic flaws will be bypassed by attacker, called broken authentication

Brute-force attacks:
- Brute-forcing usernames, passwords

- Flawed brute-force protection
+ Locking IP that have many failed attempt
+ Account loking
+ User rate limiting
- HTTP basic authentication

- Two-factor authentication tokens
- Bypassing two-factor authentication
- Brute-forcing 2FA verification codes

- Description:
+ This website have XSS injection in comment
+ Try script XSS to get target's cookie
+ Wait until victim click that page and cookie is sent to exploit server
+ In cookie have stay_logged token
+ Put token to https://hashes.com/en/tools/hash_identifier and got result token=base64(username+md5_hash(password))
+ Decode base64 and get a password_md5_hashed
+ Put it in https://crackstation.net/, if it's a popular password, this web can crack
